This directory should contain the models required by @vladmandic/face-api. Download the following models and place them here:

- face_detection_model-weights_manifest.json
- face_detection_model-shard1
- face_landmark_68_model-weights_manifest.json
- face_landmark_68_model-shard1
- face_expression_model-weights_manifest.json
- face_expression_model-shard1

You can download the models from:
https://github.com/vladmandic/face-api/tree/main/model

After downloading, your directory should look like:
public/models/
  face_detection_model-weights_manifest.json
  face_detection_model-shard1
  face_landmark_68_model-weights_manifest.json
  face_landmark_68_model-shard1
  face_expression_model-weights_manifest.json
  face_expression_model-shard1
